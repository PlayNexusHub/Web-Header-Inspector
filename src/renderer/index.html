Getting Started

                        
Enter a URL in the input field and click "Analyze Headers" to perform a security analysis of the website's HTTP headers.

                    

                    

                        
Security Score

                        
The security score is calculated based on the presence and configuration of security headers:

                        
    
                            
  • A+/A: Excellent security (80-100 points)
    • 
                            
  • B: Good security (70-79 points)
    • 
                            
  • C: Average security (60-69 points)
    • 
                            
  • D/F: Poor security (<60 points)
